## 📊 项目亮点

- **🎯 集中管理**：一台服务器统一管理所有 SSL 证书
- **⚡ 全自动化**：智能证书申请、续期、分发，零人工干预
- **🌐 多协议支持**：HTTP 与 gRPC 双协议
- **🔌 丰富生态**：Caddy、Kubernetes 等主流平台集成
- **🛡️ 高可用**：主备切换与故障转移
- **📈 易扩展**：支持多种 ACME 与 DNS 服务商
